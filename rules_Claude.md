## Rules

### 1. File naming — all files
Every file Claude creates must end with `_Claude` before the extension, regardless of file type.
**Examples:** `Summary_Claude.md`, `Architecture_Claude.md`, `Rachid_Project_Discussion_Deck_Claude.html`
This allows Jan to distinguish Claude-generated files from his own.
